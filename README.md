## Русский

Импортер для файлов Dagor Model File (`.dmf`) и Dagor Prop Layout (`.dpl`), созданных через Dagor Asset Explorer.

**Эта версия адаптирована для Blender 4.5.**

### Установка

1. Скачайте или соберите ZIP-архив аддона.
2. В Blender откройте:
   `Edit → Preferences → Add-ons → Install...`
3. Выберите архив:
   `Dagor-Asset-Explorer-Tools-Blender-4.5.zip`
4. Включите аддон в списке дополнений.
5. Импорт будет доступен через:
   `File → Import → Dagor Model File (.dmf)` или `Dagor Prop Layout (.dpl)`.

### Выбор языка в аддоне

В этой версии добавлен выбор языка интерфейса аддона:

- `English`
- `Русский`

Где переключить язык:

1. `Edit → Preferences → Add-ons`
2. Найдите `Dagor Asset Explorer Tools`
3. В настройках аддона выберите `Language / Язык`

Также выбор языка отображается в панели импорта файлов Blender.

### Возможности

- Поддержка скелета.
- Масштабирование и преобразование модели под координаты Blender/Source.
- Импорт `.dmf` моделей.
- Импорт `.dpl` layout-файлов.
- Отключение автосохранения при загрузке `.dpl`, чтобы Blender не создавал огромные временные файлы.
- Автоматическое создание shader graph для материалов.
- Поддержка текстур из папки `textures` рядом с моделью.
- Исправления совместимости с Blender 4.5.

### Исправления для Blender 4.5

Эта версия учитывает изменения Python API и shader API в Blender 4.x:

- `Principled BSDF: Specular` → `Specular IOR Level`.
- `Principled BSDF: Emission` → `Emission Color`.
- `Separate RGB` / `Combine RGB` → `Separate Color` / `Combine Color`.
- Более безопасная обработка `mesh.from_pydata(..., shade_flat)`.
- Безопасная обработка custom normals, чтобы импорт не падал на битых/нулевых нормалях.

### Если детали самолета не на своих местах

Некоторые модели Dagor, особенно самолеты, могут по-разному хранить подвижные или отделяемые детали: закрылки, шасси, пилоны, винты, элементы кокпита и т.п.

Если детали появляются в центре сцены или смещены относительно самолета, попробуйте импортировать модель заново с такими настройками:

1. Включите `Force skinned bone transforms` / `Применять трансформации костей к skinned-деталям`.
2. Если модель всё ещё повернута или смещена неправильно, дополнительно попробуйте `Legacy -90° Z rotation` / `Старый поворот Z -90°`.

Рекомендуемый порядок проверки:

```text
Вариант 1:
Fix transform = ON
Relative parenting = ON
Force skinned bone transforms = ON
Legacy -90° Z rotation = OFF

Вариант 2:
Fix transform = ON
Relative parenting = ON
Force skinned bone transforms = ON
Legacy -90° Z rotation = ON
```

### Примечание по `.dpl`

При импорте больших `.dpl` файлов Blender может создавать очень большие временные файлы. Поэтому опция `Disable auto-save` включена по умолчанию.

## English

Importer for Dagor Model Files (`.dmf`) and Dagor Prop Layout (`.dpl`) files generated by Dagor Asset Explorer.

**This fork is updated for Blender 4.5.**

### Installation

1. Download or build the add-on ZIP archive.
2. In Blender open:
   `Edit → Preferences → Add-ons → Install...`
3. Select:
   `Dagor-Asset-Explorer-Tools-Blender-4.5.zip`
4. Enable the add-on.
5. Importers will be available from:
   `File → Import → Dagor Model File (.dmf)` or `Dagor Prop Layout (.dpl)`.

### Add-on language selection

This version adds an interface language selector:

- `English`
- `Русский`

Where to change the language:

1. `Edit → Preferences → Add-ons`
2. Find `Dagor Asset Explorer Tools`
3. Change `Language / Язык`

The language selector is also shown in Blender's file import panel.

### Features

- Skeleton support.
- Model transform/scaling for Blender/Source coordinates.
- `.dmf` model import.
- `.dpl` layout import.
- Auto-save disabling while loading `.dpl` files to avoid huge temporary files.
- Automatic shader graph generation.
- Texture loading from the `textures` directory next to the model.
- Blender 4.5 compatibility fixes.

### Blender 4.5 compatibility notes

This version handles Blender 4.x Python/shader API changes:

- `Principled BSDF: Specular` → `Specular IOR Level`.
- `Principled BSDF: Emission` → `Emission Color`.
- `Separate RGB` / `Combine RGB` → `Separate Color` / `Combine Color`.
- Safer `mesh.from_pydata(..., shade_flat)` handling.
- Safer custom normals handling so imports do not fail on zero/invalid normal data.

### Troubleshooting misplaced aircraft parts

Some Dagor aircraft models can store moving/detachable parts differently: flaps, landing gear, pylons, propeller parts, cockpit details, etc.

If parts appear at the scene origin or away from the aircraft, try importing again with:

1. Enable `Force skinned bone transforms`.
2. If the hierarchy still looks rotated or offset, also try `Legacy -90° Z rotation`.

Recommended test order:

```text
Variant 1:
Fix transform = ON
Relative parenting = ON
Force skinned bone transforms = ON
Legacy -90° Z rotation = OFF

Variant 2:
Fix transform = ON
Relative parenting = ON
Force skinned bone transforms = ON
Legacy -90° Z rotation = ON
```

### DPL note

When importing large `.dpl` files, Blender may create very large temporary files. For this reason, `Disable auto-save` is enabled by default.
