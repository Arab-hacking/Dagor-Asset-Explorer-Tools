# Dagor Asset Explorer Tools

Importer for Dagor Model Files (DMF) and Dagor Prop Layout (DPL) files generated by the Dagor Asset Explorer.

*Updated fork: supports Blender 4.5*

Installation instructions are available in the [Releases tab](https://github.com/Gredwitch/Dagor-Asset-Explorer-Tools/releases).

## Features

- Skeleton support
- Model scaling to the Source engine
- Disabled auto-save when loading DPL (so you don't end up with 10GB large temporary files)
- Automatic shader graph generation
	- Pseudo-PBR magic: 50% of the normal's blue channel is multiplied onto the diffuse's alpha channel
	- Inverts the alpha channel whe necessary for camo support
	- Takes advantage of the normal map suitably:
		1) Turns the blue channel into solid white
		2) Recomposes the normal map into AGB
		3) Inverts the alpha channel and sends it to the *Specular* input of the Principled BSFD
		4) Sends the untouched blue channel into the *Metallic* input of Princinpled BSFD

## Blender 4.5 compatibility notes

This workspace version updates the add-on for Blender 4.5 by handling Blender 4.x shader API changes:

- Principled BSDF socket rename: `Specular` → `Specular IOR Level`.
- Principled BSDF socket rename: `Emission` → `Emission Color`.
- RGB node replacements: `Separate RGB`/`Combine RGB` → `Separate Color`/`Combine Color`.
- Safer mesh creation version check for the Blender 3.6+ `from_pydata(..., shade_flat)` signature.

Install the add-on in Blender 4.5 from the ZIP archive or by copying the `io_scene_daet` directory into Blender's add-ons folder.


### Troubleshooting misplaced aircraft parts

Some Dagor aircraft models can mark detachable/moving parts differently. If parts such as flaps, gear, pylons, propeller pieces or cockpit details appear at the origin or away from the aircraft, try importing again with:

1. `Force skinned bone transforms` enabled.
2. If the hierarchy still looks rotated/offset, also try `Legacy -90° Z rotation`.

These options are exposed in the DMF import transform panel. `Force skinned bone transforms` is also available for DPL imports and is passed to the DMF files loaded by the layout importer.
